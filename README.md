Chronostur es un sistema básico de gestión de clientes desarrollado en Java con PostgreSQL y JDBC, como evidencia GA7-220501096-AA2-EV01. Utiliza NetBeans 28, Maven y una estructura modular con paquetes para modelo, configuración, DAO e interfaz principal. La base de datos se llama chronostur_db y contiene la tabla "Clientes" con columnas id, nombre y telefono. El proyecto implementa operaciones CRUD completas: insertar, buscar por ID, listar todos, actualizar y eliminar. La conexión se realiza mediante DatabaseConfig.java y se captura el ID generado con RETURNING id. Al ejecutar App.java se muestra en consola el flujo completo del cliente insertado, encontrado y listado.
